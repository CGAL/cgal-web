<p>
This is a bug fix release
</p>

<h3> Number Types</h3>
<ul>
  <li> Fix MP_Float constructor which crashed for some values.</li>
</ul>

<h3> Kernel </H3>
<ul>
  <li> Rename Bool to avoid a clash with a macro in X11 headers.</li>
</ul>
<h3> Arrangement</H3>

<ul>
  <li> Derived the Arr_segment_traits_2 Arrangement_2 traits class from the parameterized Kernel.
    This allows the use of this traits class in an extended range of applications that require
    kernel objects and operations on these objects beyond the ones required by the Arrangement_2
    class itself.</li>
  <li> Fixed a compilation bug in the code that handles overlay of arrangements instantiated with
    different DCEL classes.</li>
  <li> Fixed a couple of bugs in the implementation of the Trapezoidal RIC point-location strategy</li>
</ul>

<h3> Triangulation, Alpha Shapes </H3>
<ul>
  <li> Qualify calls to filter_iterator with "CGAL::" to avoid overload ambiguities with
  Boost's filter_iterator.</li>
</ul>

<h3> Surface Mesher</H3>
<ul>
  <li>  Fixed a bug in iterators of the class template Surface_mesh_complex_2_in_triangulation_3</li>
</ul>

<h3> Surface Mesh Parametrisation </H3>
<ul>
  <li>   Updated the precompiled taucs lib</li>
</ul>
<h3> Kinetic Data Structures</H3>
<ul>
  <li> Fixed problems caused by old versions of gcc being confused by operator! and operator int()</li>
  <li> Added point removal support to the Active_objects_vector</li>
</ul>
