With a Package Manager

You can use either macports or homebrew to get CGAL on Mac OS X. 
# using macports
sudo port install cgal
sudo port install cgal +qt5 +universal +demos # with the demos

# using homebrew
brew install cgal

Build from Source on UNIX-Like Systems

To build CGAL from source on Mac OS X you have to download a release as a tarball or you can clone the current development version of CGAL from GitHub. The build process is documented in the Installation Manual.

Please note that the directory layout and build process of the development version on GitHub are different from the release version.
